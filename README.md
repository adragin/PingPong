## Пинг-понг

1. Матрица задается при старте.
2. Точка старта выбирается рандомно.
3. "Мячик" движется по диагонали.
4. Условия конца движения "мячика":
 - "мячик" попал в угол;
 - петля внутри матрицы замкнулась.
